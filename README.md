## Install

- Node.js 16+ is recommended.

### NPM

To install via NPM:

```
npm install @digitalcredentials/data-integrity-rn
```

### Development

To install locally (for development):

```
git clone https://github.com/digitalcredentials/data-integrity-rn.git
cd data-integrity-rn
npm install
```
